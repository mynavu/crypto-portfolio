# Crypto Portfolio Tracker

Track and manage crypto assets using React, Wagmi, and Alchemy. This project is a web-based crypto portfolio tracker that allows users to connect their wallet, view balances, and monitor asset performance.

## Tech Stack

- React
- TypeScript
- Vite
- Wagmi
- Alchemy
- Tailwind CSS
- Express.js

## Installation
Clone the repository and install dependencies:
```bash
git clone https://github.com/mynavu/crypto-portfolio.git
cd crypto-portfolio
npm install
npm run dev
```
On another terminal:
```bash
npx vercel dev
```

## Environment Variables

Create a `.env` file in ./backend. Paste your Alchemy API key from https://www.alchemy.com and add the following line:
```
ALCHEMY_KEY=your_api_key_here
```

## Usage

1. Connect your wallet
2. Select a network
3. View your portfolio balances
